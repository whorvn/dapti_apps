About Student Task Analysis

This tool analyzes student task performance data to identify:

  • Students who consistently work on tasks
  • Performance trends over time
  • Success rates across different subjects
  • Task completion patterns

After uploading your data, you can filter by date range, minimum success rate, and minimum working days to find the most active students.

You can also view detailed profiles for each student and compare multiple students.
